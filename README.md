> 此系統是一種用於自動搶課的程序，旨在幫助學生在選課系統開放時盡選到所需的課程。

## 功能

設置學號及密碼 **（請自行斟酌使用，建議設置好後使用「移除查看功能」避免有心人盜取學號密碼）**

```bash
學號： D08156152
密碼： 123456789
```

- [x] 自動登入
- [x] 自行選擇課程
